# Dataset Sources

## Twitter Suicidal Intention Dataset

Source:
https://github.com/laxmimerit/twitter-suicidal-intention-dataset/blob/master/twitter-suicidal_data.csv

Description:
This dataset contains Twitter posts labeled for suicidal intention detection and was used as the cross-domain evaluation dataset in this project.
